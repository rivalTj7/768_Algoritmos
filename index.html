FUNCIONES AUXILIARES REQUERIDAS

Función Descripción Parámetros Retorna
convertirABinario Convierte un número entero a su representación binaria int numero String binario
buscarProducto Busca un producto por código String codigo int posición (-1 si no encuentra)
verInventario Muestra todos los productos en formato tabla ninguno void
intercambiar Intercambia elementos en posiciones de arrays int pos1, int pos2 void

Caso Entrada Salida Esperada
Código inválido Código: 123 (menos de 5 dígitos) Error: Código debe tener 5 dígitos
Precio negativo Precio: -10 Error: Precio inválido
Stock insuficiente Stock: 5, Vender: 10 Error: Stock insuficiente
Producto duplicado Código: 10001 (ya existe) Error: Código duplicado
Inventario lleno Agregar producto #1001 Inventario lleno

📌 NOTAS IMPORTANTES PARA LOS ESTUDIANTES:

  1. Sigan EXACTAMENTE los diagramas: No agreguen ni quiten pasos. La calificación depende de que el programa haga exactamente lo que muestran los diagramas.
  2. Variables globales necesarias:
    • String[] codigos = new String[1000];
    • String[] nombres = new String[1000];
    • double[] precios = new double[1000];
    • int[] cantidades = new int[1000];
    • int[] vendidos = new int[1000];
    • int productos = 0; // contador de productos
  3. Validaciones críticas:
    • SIEMPRE validar que haya productos antes de vender/reportes/estadísticas
    • SIEMPRE validar entradas numéricas
    • SIEMPRE verificar límites de arrays
  4. Formato de salida: Respeten los formatos mostrados en los diagramas. Usen los mismos mensajes y símbolos (✓, ⚠️, etc.)
  5. Pruebas: Prueben TODOS los casos de error antes de entregar.
